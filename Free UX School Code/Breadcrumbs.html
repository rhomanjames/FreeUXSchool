What are Breadcrumbs?

By: John James

Approximate read time: 4 min

Overview

Breadcrumbs are a navigation element that allows users to understand their current location within a website or application, and how it relates to the overall hierarchy of the content. They are typically displayed as a series of links, with the current page or section represented as the last link in the series.

Overview

For example, consider a website that sells clothing. The homepage may include categories such as "men's," "women's," and "kids," with subcategories under each of these main categories. If a user clicks on the "women's" category, they may see a list of subcategories such as "dresses," "tops," and "bottoms." If the user then clicks on the "dresses" subcategory, they may see a list of individual dress products. In this case, the breadcrumb trail might be displayed as "home > women's > dresses," indicating the user's current location within the website's hierarchy.

Three Reasons Why Breadcrumbs are Essential as a UX Designer

Breadcrumbs are essential to the UX design and architecture process for several reasons.
  1. Breadcrumbs provide context and orientation for the user. They help the user understand where they are within the website or application and how to navigate back to previous pages or sections. This can be especially useful on websites with deep hierarchies or complex navigation structures.
  2. Breadcrumbs help users to discover and explore new content on the website or application. By showing the user the hierarchy of the content, breadcrumbs can help them to find related or supplementary content that they may not have otherwise discovered.
  3. Breadcrumbs are important for accessibility. They provide an additional means of navigation for users who may have difficulty using the main navigation menu, such as users with visual impairments or motor impairments. By allowing users to navigate using the breadcrumb trail, designers can improve the accessibility of their products.

Four Things to Consider When Designing Breadcrumbs

When designing breadcrumbs, it is important for UX designers to consider the following:
  1. Placement: Breadcrumbs should be placed in a consistent and visible location on the page, such as at the top or near the main navigation menu. This makes them easy for the user to find and use.
  2. Labeling: Breadcrumb links should be labeled clearly and accurately, using language that is familiar to the user. The labels should reflect the hierarchy of the content and the user's current location within it.
  3. Style: Breadcrumbs should be styled consistently and distinctively, to make them easy for the user to identify and distinguish from other elements on the page.
  4. Functionality: Breadcrumb links should be functional and clickable, allowing the user to navigate back to previous pages or sections.

Conclusion

Overall, breadcrumbs are a vital element of the UX design and architecture process. They provide context, orientation, and accessibility for the user, and help them to discover and explore new content on the website or application. By designing breadcrumbs that are clear, accurate, and easy to use, UX designers can create a more intuitive and enjoyable user experience.
